# Bolso Inteligente — TCC 3º Ano

Aplicativo mobile desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de ajudar usuários a controlarem suas finanças pessoais de forma simples e eficiente.

# Sobre o Projeto

Este projeto consiste em um aplicativo mobile para:

- Registro de receitas e despesas
- Visualização de extrato financeiro
- Acompanhamento de saldo
- Análise de gastos

# Objetivo

Permitir que o usuário tenha controle básico da sua vida financeira através de uma interface simples e intuitiva. Com automações para aumentar a praticidade e eficiência. O foco do projeto é utilizar das tecnologias que estudamos para proporcionar uma forma mais eficiente para gerir suas finanças.
